# 📻 MBC 표준FM LIVE (AU Edition)

호주에서 한국 MBC 표준FM을 끊김 없이 들을 수 있도록 최적화된 웹 앱입니다.

## ✨ 주요 기능
- **아이폰 최적화**: 홈 화면에 추가 시 실제 앱처럼 작동 (PWA)
- **버퍼링 방지**: 호주-한국 간 네트워크 지연을 고려한 자동 재연결 로직
- **듀얼 시계**: 시드니(Sydney)와 서울(Seoul) 시간을 동시에 확인
- **글래스모피즘 UI**: Apple Music 스타일의 세련된 디자인

## 🚀 사용 방법
1. Safari 브라우저에서 이 페이지의 URL에 접속합니다.
2. 하단 **[공유 버튼]**을 누릅니다.
3. **[홈 화면에 추가]**를 선택합니다.
4. 바탕화면의 아이콘을 터치하여 라디오를 감상합니다.
