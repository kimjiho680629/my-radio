# 📻 MBC 표준FM LIVE (Australia Edition)

호주에서 한국 MBC 표준FM을 끊김 없이 청취할 수 있도록 최적화된 초경량 웹 애플리케이션입니다.

## 🌟 주요 특징 (v1.4.0 업데이트 완료)

1. **울트라 안정성 (Ultra-Stability)**: 
   - **60초 안전 마진**: 라이브 엣지에서 1분 뒤의 시점을 재생하여 실시간 네트워크 흔들림에 완벽히 대응합니다.
   - **스무스-플로우(Smooth-Flow)**: 재생 시작 전 약 15~20초간 버퍼를 강제로 확보하여 초기 끊김 현상을 방지합니다.
   - **대용량 버퍼링**: 최대 3~5분 분량의 오디오 데이터를 메모리에 미리 저장하여 네트워크 단절 시에도 방송이 유지됩니다.

2. **지능형 복구 시스템 (3-Stage Recovery)**:
   - **1단계 (3초 멈춤)**: 즉시 -60초 지연 지점으로 점프하여 버퍼 재확보.
   - **2단계 (8초 멈춤)**: 세션 강제 초기화(Reload)로 '소리 반복' 현상 해결.
   - **3단계 (15초 멈춤)**: 가용 가능한 백업 서버로 즉시 전환(Failover).

3. **호주 거주자 맞춤형 UI**:
   - **듀얼 시계**: 시드니(SYD)와 서울(SEL) 시간을 실시간으로 비교 확인.
   - **데이터 모니터링**: 현재 재생 비트레이트(kbps) 및 총 데이터 사용량(MB) 표시.
   - **수동 동기화(🔄)**: 방송 상태가 이상할 때 클릭 한 번으로 스트림을 즉시 새로고침.

4. **아이폰(iOS) 최적화**:
   - Apple Music 스타일의 글래스모피즘(Glassmorphism) 다크 테마 적용.
   - 홈 화면에 추가 시 앱처럼 동작하는 PWA 기능 지원.

## 🚀 사용 방법

1. **접속**: `https://[유저아이디].github.io/my-radio` (GitHub Pages)
2. **재생**: 중앙의 재생 버튼을 누릅니다.
   - *주의: 초기 버퍼링 확보를 위해 소리가 나오기까지 약 10~20초 정도의 시간이 소요될 수 있습니다.*
3. **팁**: 방송이 끊기거나 소리가 반복된다면 하단의 🔄 버튼을 눌러주세요.

## 🛠️ 기술 사양

- **Stack**: Pure HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Library**: [HLS.js](https://github.com/video-dev/hls.js/) (비-iOS 환경 최적화).
- **Deployment**: GitHub Pages (Static Hosting).
- **Source**: 128kbps 고음질 스트림 서버 (해외 개방 서버 우선 순위).
